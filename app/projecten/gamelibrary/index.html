Gamelibrary website

In dit project was de opdracht om een gamelibrary website te maken.

De website bevatte de volgende functionaliteiten:
  • Games weergeven.
  • Games aanmaken, editen en verwijderen.
Het project was zowel uitdagend als leerzaam.

Ik heb veel geleerd over webontwikkeling en het gebruik van databases.
